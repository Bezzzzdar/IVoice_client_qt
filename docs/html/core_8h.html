◆ LIBCORE_API

#define LIBCORE_API   Q_DECL_IMPORT

Macro for exporting/importing library symbols.

This macro ensures the correct handling of symbol visibility depending on the build context:

  • If LIBCORE_EXPORTS is defined, symbols are exported (Q_DECL_EXPORT).
  • Otherwise, symbols are imported (Q_DECL_IMPORT).

Macro for logging messages with automatic context information.

This macro simplifies the process of logging by automatically including the log level, file name, line number, and function name in the log message.

Usage example:

LOG(Info) << "Application started";
